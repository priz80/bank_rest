## 🔐 Аутентификация
Аутентифицирует пользователя и возвращает JWT токен.
```POST /api/auth/login```

**Запрос**
```json
{
  "username": "string",
  "password": "string"
}****
```

**Ответ (200 OK)**
```json
{
  "token": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.x...",
  "userId": 1,
  "username": "ivan",
  "role": "USER"
}
```

**Ошибки**
- *401 Unauthorized — Неверные логин или пароль*

## 🃏 Карты (доступно для USER и ADMIN)
#### Получить свои карты с пагинацией.
```GET /api/cards```

> ⚠️ Пользователь с ролью ADMIN получит все карты всех пользователей

**Параметры**

| Параметр | Тип | По умолчанию | Описание |
|----------|-----|--------------|----------|
| page | integer | 0 | Номер страницы |
| size | integer | 10 | Размер страницы |
| sort | string | id,asc | Поле и направление: id,desc, expiryDate,asc |

**Ответ (200 OK)**
```json
{
  "content": [
    {
      "id": 1,
      "cardNumber": "1234 **** **** 5678",
      "cardHolderName": "IVAN PETROV",
      "expiryDate": "2029-01-01",
      "status": "ACTIVE",
      "balance": 5000.00,
      "userId": 1
    }
  ],
  "totalElements": 1,
  "totalPages": 1,
  "number": 0,
  "size": 10
}
```
---
#### Создать новую карту для авторизованного пользователя.
```POST /api/cards```

> ⚠️ Автогенерация номера и cvv кода, срок действия + 5лет

**Запрос**
```json
{
  "userId": 1
}
```

**Ответ (200 OK)**
```json
{
  "id": 2,
  "cardNumber": "4123567890123456",
  "cardHolderName": "IVAN PETROV",
  "cvv": "123",
  "expiryDate": "2029-01-01",
  "status": "ACTIVE",
  "balance": 0.00,
  "userId": 1
}
```
---
#### Получить карту по ID.
```GET /api/cards/{id}```

**Ответ (200 OK)**
```json
{
  "id": 1,
  "cardNumber": "1234 **** **** 5678",
  "cardHolderName": "IVAN PETROV",
  "expiryDate": "2029-01-01",
  "status": "ACTIVE",
  "balance": 5000.00,
  "userId": 1
}
```

**Ошибки**
- *404 Not Found — Карта не найдена или не принадлежит пользователю*
- *403 Forbidden — Доступ запрещён*
---
#### Заблокировать карту по ID.
```POST /api/cards/{id}/block```

**Ответ (200 OK)**
```json
{
  "id": 1,
  "status": "BLOCKED",
  "balance": 5000.00
}
```

**Ошибки**
- *400 Bad Request — Карта уже заблокирована*
- *404 Not Found — Карта не найдена*
---
#### Активировать карту по ID.
```POST /api/cards/{id}/activate```

**Ответ (200 OK)**
```json
{
  "id": 1,
  "status": "ACTIVE",
  "balance": 5000.00
}
```

**Ошибки**
- *400 Bad Request — Карта уже активна или просрочена*
- *404 Not Found — Карта не найдена*

## 👮 Администрирование (только для пользователя с ролью ADMIN)
#### Получить всех пользователей.
```GET /api/admin/users```

**Параметры**

| Параметр | Тип | По умолчанию | Описание |
|----------|-----|--------------|----------|
| page | integer | 0 | Страница |
| size | integer | 10 | Размер |
| sort | string | id,asc | Поле и направление |


**Ответ (200 OK)**
```json
{
  "content": [
    {
      "id": 1,
      "username": "ivan",
      "role": "USER",
      "status": "ACTIVE"
    }
  ],
  "totalElements": 1,
  "totalPages": 1,
  "number": 0,
  "size": 10
}
```
---
#### Получить пользователя по ID.
```GET /api/admin/users/{id}```

**Ответ (200 OK)**
```json
{
  "id": 1,
  "username": "ivan",
  "role": "USER",
  "status": "ACTIVE"
}
```

**Ошибки**
- *404 Not Found — Пользователь не найден*
---
#### Создать **нового** пользователя.
```POST /api/admin/users```

**Запрос**
```json
{
  "username": "anna",
  "password": "secret123"
}
```

**Ответ (200 OK)**
```json
{
  "id": 2,
  "username": "anna",
  "role": "USER",
  "status": "ACTIVE"
}
```

**Ошибки**
- *400 Bad Request — Пользователь с таким именем уже существует*
---
#### Изменить статус пользователя по ID.
```PUT /api/admin/users/{id}/status```

**Параметры**

| Параметр | Описание |
|----------|----------|
| status | ACTIVE, BLOCKED, DELETED |

**Ответ (200 OK)**
```json
{
  "id": 1,
  "status": "BLOCKED"
}
```

**Ошибки**
- *400 Bad Request — Некорректный статус*
- *404 Not Found — Пользователь не найден*
---
#### Удалить пользователя из БД по ID.
```DELETE /api/admin/users/{id}```

> ⚠️ Условия:
- Статус пользователя должен быть DELETED
- У пользователя не должно быть карт

**Ответ (204 No Content)**

**Ошибки**
- *400 Bad Request — Пользователь не в статусе DELETED или есть карты*
- *404 Not Found — Пользователь не найден*

#### Заблокировать карту по ID
```POST /api/admin/cards/{id}/block```

Администратор блокирует карту принудительно.

> ⚠️ Даже если карта уже активна или просрочена — можно заблокировать

**Параметры**

| Параметр | Тип | Описание |
|----------|-----|----------|
| id | Long | ID карты |

**Ответ (200 OK)**
```json
{
  "id": 1,
  "status": "BLOCKED",
  "balance": 5000.00
}
```

**Ошибки**
- *400 Bad Request — Карта уже заблокирована*
- *404 Not Found — Карта не найдена*
- *403 Forbidden — Доступ запрещён*
---
#### Активировать карту
```POST /api/admin/cards/{id}/activate```

Администратор активирует карту, даже если она была заблокирована.

> ⚠️ Не может активировать просроченную карту (EXPIRED)

**Параметры**

| Параметр | Тип | Описание |
|----------|-----|----------|
| id | Long | ID карты |

**Ответ (200 OK)**
```json
{
  "id": 1,
  "status": "ACTIVE",
  "balance": 5000.00
}
```

**Ошибки**
- *400 Bad Request — Карта уже активна или просрочена*
- *404 Not Found — Карта не найдена*
- *403 Forbidden — Доступ запрещён*
---
#### Перевод средств между любыми картами
```POST /api/admin/cards/transfers```

Администратор может переводить деньги с любой на любую карту, независимо от владельца.

**Запрос**
```json
{
  "fromCardId": 1,
  "toCardId": 2,
  "amount": 100.00
}
```

**Поля**

| Поле | Тип | Описание |
|------|-----|----------|
| fromCardId | Long | ID карты-отправителя |
| toCardId | Long | ID карты-получателя |
| amount | BigDecimal | Сумма перевода (> 0) |

**Ответ (200 OK) — пустой**

**Ошибки**
- *400 Bad Request — Недостаточно средств, сумма ≤ 0, карта не найдена*
- *404 Not Found — Карта не существует*
- *403 Forbidden — Доступ запрещён*
---
#### Получить карту по ID
```GET /api/admin/cards/{id}```

Администратор получает полную информацию о любой карте в системе.

**Параметры**

| Параметр | Тип | Описание |
|----------|-----|----------|
| id | Long | ID карты |

**Ответ (200 OK)**
```json
{
  "id": 1,
  "cardNumber": "1234 **** **** 5678",
  "cardHolderName": "IVAN PETROV",
  "expiryDate": "2029-01-01",
  "status": "ACTIVE",
  "balance": 5000.00,
  "userId": 1
}
```

🔒 Номер карты маскируется: **** **** **** 1234

**Ошибки**
- *404 Not Found — Карта не найдена*
- *403 Forbidden — Доступ запрещён*
---
#### Удалить карту по ID
```DELETE /api/admin/cards/{id}```

Администратор удаляет карту из системы.

> ⚠️ Условия:
> - Карта должна быть в статусе BLOCKED или EXPIRED
> - Баланс должен быть 0.00

## 🧩 Формат ответов

Пагинация

Все списковые ответы используют формат Page<T>:

```json
{
  "content": [...],
  "totalElements": 100,
  "totalPages": 10,
  "number": 0,
  "size": 10
}
```
---
## ❌ Коды ошибок

| Код | Описание |
|-----|----------|
| 400 Bad Request | Некорректные данные, бизнес-ограничение |
| 401 Unauthorized | Необходима аутентификация |
| 403 Forbidden | Доступ запрещён (не та роль) |
| 404 Not Found | Ресурс не найден |
| 500 Internal Server Error | Внутренняя ошибка сервера |

## 🔐 Авторизация

```

```
Токен получем при авторизации.
